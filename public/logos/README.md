# Logos

Drop institution / company / team logos here, then reference them from
`src/data/timeline.ts`, `src/data/projects.ts` or `src/data/profile.ts`.

Suggested files (referenced by the starter data — all optional, a monogram is
shown automatically when a logo is missing):

- `icai-icade.png`
- `boston-university.png`
- `smu.png`
- `icai-rocket-team.png`
- `placeholder-logo.svg` (included)

Tip: square logos (PNG/SVG with transparent background) look best in the
rounded glass badges.
