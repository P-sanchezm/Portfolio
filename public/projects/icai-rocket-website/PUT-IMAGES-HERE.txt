ICAI ROCKET TEAM WEBSITE — images for this project
===================================================
Put these files in THIS folder (public/projects/icai-rocket-website/):

  cover.png         -> main card image      (recommended ~1280x800, 16:10)
  screenshot-1.png  -> gallery image 1      (16:10)

NOTES
- File NAMES must match exactly (lowercase, .png).
- Any pixel size works: images are auto-cropped to fit, never stretched.
- PNG, JPG and WebP all work. To use a .jpg instead of .png, change the
  matching path in src/data/projects.ts.
- You can delete this .txt file once your images are in (optional).
