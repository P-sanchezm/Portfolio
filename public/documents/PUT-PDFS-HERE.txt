DOCUMENTS (PDFs)
================
Put PDF files in THIS folder (public/documents/) with these names:

  pablo-cv.pdf                 -> your CV (hero + navbar + contact buttons)
  cervical-airbag-report.pdf   -> linked from the Cervical Airbag project
  carbon-capture-report.pdf    -> linked from the Carbon Capture project

Add more PDFs (any name) and link them from a project's `documents` array in
src/data/projects.ts:
  documents: [{ title: "Final report", file: "/documents/your-file.pdf" }]

NOTES
- File names must match the paths used in src/data/*.ts.
- You can delete this .txt file once your PDFs are in (optional).
